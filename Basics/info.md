## 强类型语言
> 通俗的定义：强类型语言一旦定义是不允许改变变量的数据类型，除非进行强制类型转换
``` js
// 定义一个名为num的变量，它的类型为int类型，值为1；
int num = 1;
boolean flag = true; 
num = flag; // Error
```

## 静态类型语言
> 静态类型语言: 在编译阶段确定所有变量的类型

> 动态类型语言: 在执行阶段确定所有变量的类型

静态类型语言 | 动态类型语言
-|-
 对类型极度严格 | 对类型非常放松 
 立即发现错误 | BUG 可能隐藏隐藏数月甚至数年
运行时性能好 | 运行时性能差
自文档化 | 可读性差
